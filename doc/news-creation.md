# How to create news in proper order

At the start of the news file, add this `Frontamatter`

```
---
title: Introduction 2
date: YYYY-MM-DD
---
```

- MM: 01 to 12
- DD: 01 to 31

# Add images

To add images, add images in the `public/img/` folder then you can add them using `![my image](/image.png)` in your .md file
